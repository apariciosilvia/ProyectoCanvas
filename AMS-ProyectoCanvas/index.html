¿Qué es Canvas?

Canvas es un elemento de HTML5 que funciona como un lienzo digital donde puedes dibujar gráficos, animaciones y manipular imágenes utilizando JavaScript. Es ideal para crear contenido visual dinámico, como juegos o visualizaciones de datos.

¿Cómo funciona?

Añade un lienzo en tu HTML:

<canvas id="lienzo" width="500" height="300"></canvas>

Accede al lienzo con JavaScript y usa un "pincel":

                
const canvas = document.getElementById('lienzo');
const ctx = canvas.getContext('2d');

Dibuja formas y texto (ejemplo sencillo):


// Dibujar un rectángulo azul
ctx.fillStyle = '#3498db';  // Color azul
ctx.fillRect(50, 50, 200, 100);  // (x, y, ancho, alto)
    
// Dibujar un círculo rojo
ctx.beginPath();  // Iniciar un nuevo dibujo
ctx.arc(350, 100, 50, 0, Math.PI * 2, false);  // (x, y, radio, ángulo inicial, ángulo final)
ctx.fillStyle = '#e74c3c';  // Color rojo
ctx.fill();  // Rellenar el círculo
ctx.closePath();  // Finalizar el dibujo
            
//FIRMO EL CUADRO
const margin = 20;
const text = 'Firmado Silvia Aparicio Martín';
ctx.fillStyle = '#000';
ctx.font = '16px Arial';
